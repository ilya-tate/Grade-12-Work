# Menu API

## GET

- ### API Endpoint
    GET `localhost:3000/api/v1/`

- ### Find by "id"
    GET `/api/v1/find/id/:id`

- ### Find by "category"
    GET `/api/v1/category/:category`

- ### Sort by "price"
    GET `/api/v1/sort/price/query?`

    #### Parameters

    - `dir=asc` or `dir=ascending`
    - `dir=dec` or `dir=descending`
